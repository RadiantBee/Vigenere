Entry point of the application. 

This program supports three modes of operation\+:
\begin{DoxyItemize}
\item Encryption\+: Encrypts the input file using a specified key.
\item Decryption\+: Decrypts the input file using a specified key.
\item Breaking\+: Attempts to break the encryption on the input file without a key.
\end{DoxyItemize}
